## Recommendation

Ship `appicon-final-recommendation.svg` for v1.0. The tarsier is more ownable than a koala, its oversized eyes stay recognizable at app-icon sizes, and the document/caret keeps the product category clear. Use `mark-02-caret.svg` as the fallback small-size system mark for favicon, tray, monochrome, and toolbar contexts.
